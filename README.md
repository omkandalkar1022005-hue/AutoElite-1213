AutoElite is a modern, responsive web application for buying and selling vehicles online.

Key Features:

Multi-language support - English, Hindi, Marathi

10+ vehicle categories - Two Wheelers, Four Wheelers, Trucks, Dumpers, Trailers, Earthmovers, Heavy Machines

Interactive shopping cart - Add/remove items with real-time updates

Advanced search - Search by name, brand, or category

Top trending models - Showcase of 4 premium vehicles

User authentication - Sign In/Sign Up with demo credentials

Responsive design - Works on mobile, tablet, and desktop

Toast notifications - Instant feedback for user actions

Category filtering - Quick tabs for easy navigation

Sticky header - Always accessible navigation

Technologies: HTML5, CSS3, JavaScript (ES6), Font Awesome Icons, Google Fonts

Target Users: Vehicle buyers, sellers, and enthusiasts looking for a seamless online vehicle marketplace experience.

Purpose: To provide a complete, interactive demo of an e-commerce platform specifically designed for the automotive industry. 🚀
